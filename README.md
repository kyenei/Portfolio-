# Portfolio
My Portfolio in Web System

**Quien Garnet's Portfolio**

I am a 3rd-year Bachelor of Science in Information Technology (BSIT) student passionate about creating beautiful, accessible, and user-centered digital experiences. This portfolio website showcases my academic projects, technical skills, and educational background, reflecting my growth and dedication as an aspiring IT professional.

With a strong interest in web development and software design, I strive to write clean, efficient code while applying thoughtful design principles to build functional and engaging digital solutions. Through this portfolio, I aim to share my work, highlight my achievements, and demonstrate my commitment to continuous learning and innovation in the field of technology.

This portfolio website was built using HTML, CSS, and JavaScript. HTML provides the website structure, CSS3= handles the styling and responsive design, while JavaScript adds interactive features. The site is hosted and deployed using GitHub Pages.
